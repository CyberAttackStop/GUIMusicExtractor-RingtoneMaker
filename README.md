# GUI Music Extractor & Ringtone Maker

GUI Music Extractor & Ringtone Maker is a professional desktop application built with Python for extracting audio from videos, separating vocals and instrumentals using AI, playing multimedia files, and creating custom ringtones with an interactive waveform editor.

The application combines FFmpeg, VLC Media Player, Demucs AI, and CustomTkinter to provide a fast, modern, and user-friendly multimedia workflow.

## Features

- Extract audio from video files
- Convert media using FFmpeg
- AI-powered Vocal & Instrument Separation (Demucs)
- Interactive waveform ringtone editor
- Drag-and-select ringtone trimming
- Live ringtone preview
- Audio metadata detection (FFprobe)
- Media player with VLC integration
- Professional CustomTkinter interface
- Automatic project folder management
- Progress tracking and logging
- Export ringtones in multiple formats
