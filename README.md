# 🎵 GUIMusicExtractor+RingtoneMaker

A modern desktop application built with Python and CustomTkinter.

## Features

- 🎬 Video Player
- 🎵 Audio Player
- 🎧 MP3 Extraction
- ✂️ Ringtone Creator
- 🖼 Thumbnail Generator
- 📊 Media Information
- 🎚 Volume Control
